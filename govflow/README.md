# GovDesk

GovDesk 是一个面向政务公众服务场景的 Skill，用来把群众留言、咨询问题和投诉诉求，转成可直接使用的分流建议、标准答复和督办口径。

## 它做什么
- 识别留言类型和紧急程度
- 提炼事实要素和关联政策
- 生成标准化答复草稿
- 给出转办和督办建议

## 适合什么时候用
- 热线、信箱、政务平台留言需要快速分流时
- 群众咨询多、需要统一回复口径时
- 投诉诉求需要判断是否紧急和是否转办时
- 需要把零散留言整理成工单信息时

## 使用建议
先提供真实留言和已有办理线索，再让 Skill 做分流和起草。涉及政策依据、时限、责任单位和办理结果时，优先补齐来源，不要让模型猜。
