# 财报指标风险洞察助手

这是一个面向金融材料智能解析与风险洞察赛题的 Skill，用来把财报、公告、研报和财务表格整理成可复核的结构化指标表、风险提示和简短结论。

## 它做什么
- 提取收入、利润、现金流、负债、应收账款、存货等关键指标
- 识别同比/环比异常和风险信号
- 输出证据摘录和一句话结论
- 在信息不足时先提示缺口，不编造数字

## 适合什么时候用
- 财报速读
- 公告重点整理
- 风险排查
- 投研摘要整理
- 信贷初筛

## 使用建议
先提供原文、表格或 OCR 文本，再让 Skill 抽取指标和判断风险。若缺少报告期、口径、对标基准或原始数字，应先补齐再做结论。
