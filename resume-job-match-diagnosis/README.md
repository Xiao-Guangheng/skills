# 简历岗位匹配诊断助手

这是一个面向人才匹配与发展赛题的 Skill，用来把简历、岗位 JD、作品集和面试记录整理成可执行的匹配度诊断和改写建议。

## 它做什么
- 解析岗位硬门槛和核心职责
- 提炼候选人亮点和项目证据
- 判断匹配度、缺口和风险
- 输出简历优化和面试准备建议

## 适合什么时候用
- 简历初筛
- 岗位匹配判断
- 简历优化
- 面试准备
- 职业规划或能力盘点

## 文件结构

resume-job-match-diagnosis/

## 使用建议
先提供简历和目标岗位，再让 Skill 做匹配诊断。若缺少职级、城市、薪资或岗位说明，应先补齐后再给最终结论。
