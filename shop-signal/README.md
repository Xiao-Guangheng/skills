# ShopSignal

ShopSignal 是一个面向电商场景的 Skill，用来把商品信息、用户评论、竞品摘要和客服问题整理成可直接落地的内容包。

## 它做什么
- 生成商品详情页文案
- 提炼用户评价中的好评点、差评点和需求点
- 给出客服回复建议和 FAQ
- 输出上新优化建议

## 适合什么时候用
- 商品上新前
- 详情页转化不高时
- 用户评论很多但没人总结时
- 客服问答重复、需要标准话术时
- 想快速判断商品卖点和风险时

## 使用建议
先提供真实商品信息和评论，再让 Skill 做卖点拆解和内容生成。缺少价格、规格、库存或功效时，先补信息，不要直接编造。
