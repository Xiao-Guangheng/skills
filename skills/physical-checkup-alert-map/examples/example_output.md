# 示例输出

- 材料类型判断：年度体检报告摘要
- 关键信息摘要：36 岁男性，空腹抽血，近期未服药
- 异常指标清单：
  - 总胆固醇 6.8 mmol/L，偏高
  - 低密度脂蛋白胆固醇 4.2 mmol/L，偏高
  - ALT 56 U/L，轻度升高
  - 尿酸 512 umol/L，偏高
  - 空腹血糖 5.9 mmol/L，接近上限，需要结合参考范围和复查情况判断
- 分层解读：
  - 轻度关注：ALT 轻度升高，常见于近期饮食、饮酒、运动、脂肪肝或短期波动，但需结合复查
  - 建议复查：总胆固醇、低密度脂蛋白胆固醇和尿酸均偏高，建议结合生活方式管理后复查
  - 尽快就医：当前摘要里未见明显急危重红旗，但若伴随胸痛、黄疸、明显乏力、持续腹痛或异常加重，应及时线下就诊
- 可能的常见解释：血脂偏高、尿酸偏高和轻度肝酶升高常与饮食结构、饮酒、体重、代谢状态相关，但不能据此直接下诊断
- 复查建议：
  - 先补充完整参考范围和既往历史
  - 1 到 3 个月内复查血脂、肝功能和尿酸
  - 如有脂肪肝、高血压、糖尿病家族史或心血管风险，建议同步门诊评估
- 生活方式提示：少油少酒、控制高嘌呤饮食、增加运动、管理体重
- 红旗提醒：如体检原文写明“建议尽快就医”或出现明显症状，应优先线下处理
- 待确认项：血压、BMI、肝胆影像结果、家族史、参考范围原文
