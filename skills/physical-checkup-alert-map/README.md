# 体检异常分层解读助手

这是一个面向医疗健康智能理解与辅助决策赛题的 Skill，用来把体检报告、检验单和指标表整理成可复核的异常清单、分层解读和复查建议。

## 它做什么
- 抽取体检中的异常指标
- 按轻重缓急分层解读
- 输出复查建议和生活方式提示
- 在信息不足时先提示补充，不硬猜结论

## 适合什么时候用
- 年度体检复盘
- 指标偏高偏低时想快速看懂
- 体检单太长、想先抓重点时
- 复查前先整理问题清单时
- 做健康管理跟进时

## 使用建议
先提供体检报告原文、截图或指标表，再让 Skill 进行分层解读。若缺少年龄、性别、空腹状态、用药情况或关键原文，先补齐再下结论。
