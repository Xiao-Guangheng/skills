# 多源证据检索核验助手

这是一个面向多源异构信息检索分析赛题的 Skill，用来把分散在文档、网页摘录、表格和知识库中的信息整理成可溯源的答案、证据矩阵和冲突提示。

## 它做什么
- 改写查询意图，帮你找对关键词
- 从多份材料中抽取证据
- 对比一致点和冲突点
- 输出带引用的答案和待确认项

## 适合什么时候用
- 跨文档查证
- 政策或项目资料检索
- FAQ 和材料问答
- 长文档定位
- 多来源信息对比

## 文件结构
```text
multi-source-evidence-check/
├─ SKILL.md
├─ README.md
├─ examples/
├─ references/
├─ scripts/
└─ templates/
```

## 使用建议
先给出问题和候选材料，再让 Skill 做证据抽取和冲突比对。若来源、时间或版本不明确，应先补充，再输出最终结论。
