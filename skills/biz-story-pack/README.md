# Biz Story Pack

Biz Story Pack 是一个面向内容创作和业务表达场景的 Skill，用来把真实业务观察、临时灵感、口述记录和历史稿件，整理成可直接开口讲、可发布、可复盘的内容包。

## 它做什么
- 判断一条内容值不值得讲
- 提炼受众、主张和结构
- 生成标题、开场、口播稿和字幕钉子
- 输出证据清单、发布前 QA 和复盘卡

## 适合什么时候用
- 你有一个业务观点，但不知道值不值得讲
- 你想把零散灵感整理成演讲或口播
- 你需要一套能复用的内容生产流程
- 你要让内容更像真实表达，而不是说明文

## 文件结构
```text
biz-story-pack/
├─ SKILL.md
├─ README.md
├─ examples/
├─ references/
├─ scripts/
└─ templates/
```

## 使用建议
先给真实观察或口述材料，再让 Skill 判断选题和生成内容。涉及数据、证据、经历和结论时，优先补足来源，不要直接让模型编造。
