# 会议行动追踪办公助手

> 面向「智能办公协同助理」赛道的参赛 Skill

一个按场景路由的办公协同 Skill，把会议转写、聊天记录、邮件和任务清单整理成可直接使用和跟踪的行动项台账与周报草案。

---

## 核心能力

### 1. 会议纪要结构化
- 从会议转写和笔记中抽取主题、参与人、核心决议
- 按"基本信息 → 核心决议 → 行动项"三段式组织输出
- 不编造会议内容，不忽略关键分歧

### 2. 行动项提取与台账
- 每条行动项必须提取三个字段：做什么、谁来做、计划完成时间
- 按状态分入"待启动 / 进行中 / 已完成 / 已逾期 / 有阻塞"看板
- 字段不完整时，用选择题追问而非硬补全

### 3. 到期预警
- 单独列出逾期和即将到期的项
- 标记高风险（已逾期）和关注（本周内到期）
- 标注阻塞项和被依赖项

### 4. 周报草案生成
- 基于已完成行动项和进展更新生成草案
- 按"本周成果 → 进行中 → 问题与风险 → 下周计划"四段组织
- 生成前确认已完成的行动项清单是否准确

### 5. 多输入场景路由
- 自动识别输入类型（会议转写/聊天记录/邮件/任务清单/周报素材）
- 不同输入类型对应不同输出格式
- 聊天记录和邮件输出 TODOs 汇总；会议输出结构化纪要+台账

## 适合什么时候用

| 场景 | 说明 |
| --- | --- |
| 会后整理 | 录音转写或会议笔记 → 行动项台账 |
| 聊天记录扫描 | 钉钉/飞书/企微/微信中的任务提取 |
| 任务状态跟进 | 到期预警 + 阻塞项排查 |
| 周报复盘 | 基于行动项生成周报草案 |
| 跨团队协同 | 明确责任人和截止时间，减少遗漏 |
| 老板追问进展 | 快速汇总当前状态和风险点 |

## 使用建议

1. 先提供会议转写或聊天记录，Skill 自动识别类型
2. 确认行动项的关键字段完整后再生成台账
3. 到期预警按优先级逐条处理，先解决高风险的阻塞项
4. 周报草案需要人工确认已完成项是否准确

⚠️ **AI 生成内容提醒**：本技能包的纪要、行动项和周报草案均由 AI 生成，可能存在信息遗漏或理解偏差。涉及关键节点和跨团队承诺务必人工确认。
