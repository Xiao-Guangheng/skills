# 会议行动追踪办公助手

## 何时使用此 Skill

用于写内部工作协同类内容时使用此 Skill：
- 会议纪要整理
- 行动项提取与跟踪台账
- 周报草案生成
- 任务状态汇总
- 工作日志整理

## 如何使用此 Skill

要处理任何协同办公类内容：

1. **识别输入类型** 从用户提供的材料判断：
   - `会议转写` — 录音转文字、会议笔记
   - `聊天记录` — 钉钉/飞书/企微/微信群聊
   - `邮件合集` — 邮件摘要或转发链
   - `任务清单` — ToDo、JIRA/Trello 导出
   - `周报素材` — 日报、工作日志、进展汇总

2. **加载相应模板** 从 `templates/` 目录加载：
   - `templates/action_tracker_template.md` — 行动项跟踪台账模板

3. **按模板生成** 严格按照模板格式输出，不要自由发挥。

## 输出格式（按场景路由）

### 如果输入是会议转写

```
## 会议基本信息
- 主题：[会议主题]
- 时间：[日期]
- 参与人：[部门/姓名]

## 核心决议
1. [决议一]
2. [决议二]

## 行动项台账
| 行动项 | 负责人 | 计划完成 | 状态 | 备注 |
| --- | --- | --- | --- | --- |
| [做什么] | [谁] | [日期] | [待启动/进行中/已完成/已逾期/阻塞] | [依赖/风险] |

## 到期预警
- 🔴 高风险：[逾期或即将到期的项]
- 🟡 关注：[本周内到期]
```

### 如果输入是聊天记录/邮件

```
## 待办事项汇总
按来源分类：

**来自 [群聊名称/邮件主题]**
- [ ] [任务] — [负责人] — [截止日期]
- [ ] [任务] — [负责人] — [截止日期]

## 缺失信息
- [ ] 需要确认：XXX 的截止日期
- [ ] 需要确认：YYY 的负责人
```

### 如果用户要求生成周报

```
## 本周成果
1. [已完成项1]
2. [已完成项2]

## 进行中
1. [进行中项1] — 进度：[X%] — 预计完成：[日期]
2. [进行中项2] — 进度：[X%] — 预计完成：[日期]

## 问题与风险
- [问题1] — 影响：[说明] — 需要支持：[说明]
- [问题2] — 影响：[说明] — 需要支持：[说明]

## 下周计划
1. [计划1]
2. [计划2]
```

## 交互规则

当行动项的关键字段（做什么、谁来做、计划完成时间）缺失时：
- 不要编造补全
- 先输出已填好的行动项
- 用选择题追问缺失字段："以下哪项最接近实际情况？"
- 每轮最多3个问题

## 字段清洗规则

输出到表格时：
- `=` `+` `-` `@` 开头的值加前置 `'`
- `|` 在 Markdown 表中转义为 `\|`
- 多行折叠为单行，用 ` / ` 分隔
- HTML 片段转义处理
- 手机号掩码中间四位、邮箱掩码 @ 前三位后、身份证保留首六尾四
- 处理顺序：脱敏 → 公式转义 → HTML转义 → Markdown转义 → 换行折叠

## AI 生成内容提醒

本技能包的纪要、行动项和周报草案均由 AI 生成，可能存在信息遗漏或理解偏差。请勿直接将输出作为正式工作记录或唯一依据；涉及关键节点和跨团队承诺务必人工确认后使用。
